# PERCEPTRON-RKMVERI (Linguine AI)

Linguine AI is a hackathon project designed to bridge language barriers in global gastronomy. 

## Project Structure

- `frontend/`: Next.js 14 application containing the UI and business logic.
- `backend/`: Placeholder for future API integration (empty as requested).

## Getting Started

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Features
- **Startup Landing Page**: A premium hero section describing the mission of Linguine AI.
- **Menu Translation Animation**: A custom CSS/Framer-Motion animation on the home page that demonstrates scanning a foreign menu and translating it in real-time.
- **Multi-Step Flow**: Seamless transition from landing to upload to results.
- **Premium Glassmorphism**: Clean, modern UI with floating navbars and glowing orbs.
